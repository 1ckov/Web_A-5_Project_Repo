Bienvenue dans notre GEZ-help!

Vous avez demandé de l'aide pour un document gez.

GEZ est la redevance de diffusion ici en Allemagne. Chaque appartement doit le payer, donc il n'y a pas d'échappatoire, si vous voulez vivre en Allemagne.

Vous devez également être âgé d'au moins 18 ans pour payer les frais de licence.

Après avoir soumis le formulaire, vous devrez payer les frais régulièrement.

Bien que cela puisse être ennuyeux de payer, vous ne devez remplir le document qu'une seule fois, tant que vous vivez dans votre appartement actuel.

Les médias publics sont soutenus par cet argent. Ces médias sont indépendants des entreprises privées qui veulent manipuler la portée du droit public à leurs propres fins.

Nous divisons le document en plusieurs petites parties.

  • Tout d'abord, nous avons besoin de vos informations générales, telles que votre nom ou votre date de naissance
  • Deuxièmement, nous avons besoin d'informations sur votre adresse
  • Ensuite, nous devons connaître votre mode de paiement préféré.
  • Si vous souhaitez utiliser SEPA, nous avons besoin de vos informations bancaires. Sinon, vous devez envoyer vous-même l'argent.
  • Enfin, si vous avez terminé, il vous suffit de signer le document et vous avez terminé!

Si vous souhaitez commencer maintenant, appuyez simplement sur le bouton de démarrage.
